    Instructions

    Welcome to JSpardy! This game is an online version of the game show, Jeopardy!. The rules are simple.
    • Select your categories. You may have as many or as few as you'd like.
    • Select either one or two players and the game board will appear.
    • Select clues and, for correct answers, you will receive the point value on the card. Incorrect answers will deduct that value from your score. Submit a blank answer to skip with no penalty.
    • If you feel your answer was misjudged as incorrect, click the "Dispute Answer" button to receive your points back.

    Info

    Thanks for playing!
    • JSpardy! was developed using HTML5, CSS/SCSS, and vanilla JavaScript.
    • Every clue, answer, and category is pulled from an API housing over 150,000 past Jeopardy! questions. You can check out the jService API here.
